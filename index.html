【社政篇】人權與民主制度
◎ 三代人權的進展 (重點複習) +
階段特色主體性質與主張
第一代消極人權個人公民自由、政治權；要求國家降低干預
第二代積極人權群體經濟、社會、文化權；要求國家積極干預 (社會福利)
第三代集體人權全人類環境、發展、民族自決權；反對歧視與隔離
◎ 西方民主理論比較 (審議 vs 菁英) +
  • 菁英民主理論 (熊彼得)： 民主是一種「選舉政治領導人」的方式，強調投票 。
  • 參與民主理論 (道爾)： 權力應高度分散，鼓勵不同勢力參與決策 。
  • 審議民主理論 (哈伯瑪斯)： 討論中心，強調公民在公平條件下理性溝通形成共識 。
◎ 民主國家政府體制 (內閣、總統、雙首長) +

左右共治 (法國)： 當國會多數黨與總統政黨不同時，總統依慣例任命反對陣營人士為總理，分享行政權 。

體制行政首長向誰負責有無倒閣/解散權
內閣制內閣向國會負責 皆有
總統制總統向全民負責 皆無
雙首長制總理向國會負責 皆有
【法律篇】基本權、刑事與民法
◎ 刑事責任能力 (年齡分界點) +
  • 無責任能力： 未滿 14 歲 (不罰) 。
  • 限制責任能力： 14~18 歲、80 歲以上 (得減輕其刑) 。
  • 完全責任能力： 18~80 歲 (負完全刑責) 。
◎ 比例原則 (三子原則) +
  1. 適當性： 手段必須有助於達成目的 。
  2. 必要性： 多種手段中，選擇對人民權利「侵害最小」者 。
  3. 衡量性： 損害不得與達成的利益顯失均衡 。
【經濟篇】生產可能曲線與 GDP
◎ GDP 與 GNP 的區分 +
  • GDP (國內生產毛額)： 以國境為界。公式：C + I + G + (X - M) 。
  • GNP (國民生產毛額)： 以國籍為界。公式：GDP + 國外要素所得淨額 。
◎ 公共財與私有財 (排他性 vs 共享性) +
排他性無排他性
獨享性純私有財 (如：手機) 準私有財 (如：海洋漁產)
共享性準公共財 (如：捷運) 純公共財 (如：國防、燈塔)
